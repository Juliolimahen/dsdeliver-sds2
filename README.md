# Como executar o projeto

## Back end - Java

Pré-requisitos: Java 11

```bash
# clonar repositório
git clone https://github.com/Juliolimahen/dsdeliver-sds2

# entrar na pasta do projeto back end
cd back-end

# executar o projeto
./mvnw spring-boot:run
```

## Back end - C# Net Core

Pré-requisitos: dot 6

```bash
# clonar repositório
git clone https://github.com/Juliolimahen/dsdeliver-sds2

# entrar na pasta do projeto back end
cd back-end-dot-net
cd DsDeliveryApi
cd DsDelivery.WebApi

# executar o projeto
dotnet run
```

## Front end web

Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone https://github.com/Juliolimahen/dsdeliver-sds2

# entrar na pasta do projeto front end web
cd front-web

# instalar dependências
yarn install

# executar o projeto
yarn start
```
